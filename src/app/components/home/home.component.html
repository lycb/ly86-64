LY86-64

The LY86-64 ("lee 86-64") is a browser-based simulator developed to help students visualize the Y86-64 machine.

Although inspired by other Y86-64 simulators, the LY86-64 will be more focused on the visualization aspect of the control logic and signals, specifically stalling and bubbling.

This is a senior thesis project built by Jenny Ly from Appalachian State University

RECOMMENDATIONS

While the LY86-64 works on any browsers, it is recommended to use Chrome for the best experience.

We recommend your browser size is at least 700x827.

The LY86-64 focuses on the visualization aspect of the control logic and signals. To do this, we have incorporated the elements and colors we think are the most important. Please read through the documentations below before using the simulator.

DROPDOWN

The dropdown presents several options:

  • Custom files: Upload your own .yo file. Currently our simulator does not check for errors within your custom files, so please be sure to upload a valid .yo file.
  • "jl", "jecall", and "loaduse": Sample files that is already loaded into the simulator. Selecting these options will not require you to upload anything to use our simulator.

    • CONTROL BUTTONS

    The control buttons allow you to control the simulator.

  • Continue: Run through the instructions until the end of the file.
  • Step: Run instruction by instruction.
  • Reset: Reset the simulator back to the first instruction.

    • CODE BLOCK

    As you step through the instructions, the line getting highlighted is the line that is going to get fetched next.

  • This colored highlight is showing that the simulator is fetching normally.
  • This colored highlight is showing that the simulator is fetching the same instruction again (i.e. the F register stalled).

    • CONTROL LOGIC

    The control logic component is designed to help the user understand why a certain register stalled or bubbled.

    PIPELINE REGISTERS

    Alongside the indicated text next to registers like F (STALL) or E (BUBBLE), the pipeline registers have several colors to show the state of the registers as well:

  • Green for when the pipeline is normal.
  • Yellowfor when the pipeline is stalled.
  • Orangefor when the pipeline is bubbled.

    • OTHER ELEMENTS

  • Registers: Shows the state of the registers during the simulation. Registers are displayed from register 0 (%rax) to register 15 (%r14).
  • Control flags: OF is the overflow flag, SF is the sign flag, ZF is the zero flag. If the flag is 0, it's false. If it's 1, then it's true.
