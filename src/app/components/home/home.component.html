LY86-64

The LY86-64 ("lee 86-64") is a browser-based simulator developed to help students visualize the Y86-64 machine.

Although inspired by other Y86-64 simulators, the LY86-64 will be more focused on the visualization aspect of the control logic and signals, specifically stalling and bubbling.

This is a senior thesis project built by Jenny Ly from Appalachian State University

Let's first go over some elements:

DROPDOWN

The dropdown presents several options:

  • "Custom files": Upload your own .yo file to run the simulator with
  • "jl", "jecall", and "loaduse": Sample files that is already loaded into the simulator.

    • CONTROL BUTTONS

    These three buttons are the main controls of the simulator. "Continue" will run through the instructions until the end of the file, "Step" will run instruction by instruction, and "Reset" will reset the simulator back to the first instruction.
